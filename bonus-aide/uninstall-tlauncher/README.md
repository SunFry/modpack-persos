# Désinstaller TLauncher
Cette méthode pour désinstaller TLauncher n'a pas été prouvée avec certitude, mais c'est actuellement celle qui a la meilleure chance de supprimer le virus. Pour faire ce tutoriel, nous nous sommes basés sur toutes les informations que nous avons obtenues grâce à diverses analyses dans Triage, AnyRun, FileScan et VirusTotal, et nous avons analysé à la fois l'installateur (binaire original) et le binaire qui vous est téléchargé.

## Étapes pour désinstaller TLauncher
Pour désinstaller TLauncher, nous utiliserons le [logiciel Revo Uninstaller](https://download.revouninstaller.com/download/revosetup.exe) ([cliquez ici pour télécharger la version portable](https://download.revouninstaller.com/download/RevoUninstaller_Portable.zip)), qui est un désinstalleur gratuit qui vous permet de désinstaller des programmes et de supprimer tous les fichiers restants. Pour ce faire, nous suivrons les étapes suivantes :
1. Tapez TLauncher dans la barre de recherche
2. Cliquez avec le bouton droit et cliquez sur « Désinstaller »
3. Décochez « Créer un point de restauration système » (Optionnel) et continuez
4. Continuez avec les étapes normales pour désinstaller TLauncher de manière classique
5. Après avoir terminé la désinstallation, sélectionnez analyse avancée et cliquez sur « Analyser », cela peut prendre quelques minutes
6. Une fois l'analyse terminée, cliquez sur le bouton « Sélectionner tout » pour supprimer toutes les modifications du pare-feu et cliquez sur « Supprimer », cela supprimera toutes les règles de pare-feu créées par TLauncher
7. Faites la même chose pour les fichiers et répertoires et cliquez sur « Supprimer », cela supprimera tous les fichiers et répertoires restants. Attention : cela peut supprimer les fichiers de .minecraft, alors soyez prudent
8. Appuyez sur Win + r et tapez « appdata » et appuyez sur Entrée, cela ouvrira le répertoire appdata
9. Accédez à LocalLow et supprimez le fichier Oracle (Cela supprimera Java, réinstallez-le si vous souhaitez l'utiliser à nouveau)
10. Utilisez MalwareBytes pour analyser les virus et supprimez tous les virus trouvés (Optionnel mais recommandé)
11. Redémarrez votre ordinateur et vous aurez désinstallé TLauncher avec succès.

8. Press Win + r and type "appdata" and press enter, this will open the appdata directory
9. Go to LocalLow and delete the file Oracle (This will delete java, reinstall if you want to use it again)
10. It uses MalwareBytes to scan for viruses and removes all viruses it finds (Optional but recommended)
